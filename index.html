GAZETTEER

Overview
Create a "mobile first", responsive website that will provide profiling for all countries through the presentation of demographic, climatic, geographical and other data. The system should make extensive use of third party APIs. Users can select any country on the map to retrieve information about that country.

Summary
This is a one-page, full screen, mobile-first web application. Javascript library Leaflet, was used to build a mobile-friendly, interactive map. Geolocation and numerous third party APIs such as, geonames and restcountries.eu were used to display countries demographic, climatic, geographical as well as historical data and live news updates.

EMPLOYEE DIRECTORY

Overview
Design a user-friendly, responsive CRUD web application that works equally well on all devices. Create an employee directory that retrieves all existing employees from a database and displays their details (name, location, department). Users should be able to successfully modify employee details as well as add, update and delete employees, departments and locations.

GATSBY BLOG

For this project, a blazing fast 4 page blog site was created using the React-based framework Gatsby along with Contentful CMS. Some data was also collected from Markdown. The website gives an app-like experience with no page reloads between pages for a smooth user experience. Site owners can also easily make changes and updates to the site using Contentful CMS. Plugins gatsby-image and gatsby-plugin-sharp were used for speedy and responsive images. Other useful plugins such as gatsby-plugin-sass for styling, gatsby-transformer-remark and gatsby-plugin-react-helmet were used to create the site.
